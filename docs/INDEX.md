## 🚀 快速命令参考

### 运行游戏
```bash
# CLI模式
python rulek.py cli

# Web模式
python rulek.py web

# 演示模式
python rulek.py demo
```

### 运行测试
```bash
# 运行所有测试
python rulek.py test

# 只运行单元测试
python rulek.py test unit

# 运行集成测试
python rulek.py test integration
```

### 环境验证
```bash
python rulek.py verify
```
